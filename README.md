# tabgraph

DBのテーブル・カラムにメタデータ（説明文）を付与し、全文検索できるツールです。
Go製の単一バイナリで動作し、ブラウザでUIを操作します。

## できること

- PostgreSQL / MySQLに接続し、テーブル・カラム構造を取得
- テーブル・カラムに説明文（メタデータ）を付与・編集
- FTS5による全文検索（テーブル名・カラム名・説明文を横断検索）
- 複数DB接続の切り替え
- ER図表示
- 単一バイナリ配布（Go + 埋め込みNext.js静的ビルド）

## セットアップ・実行

### 開発環境

```bash
# APIサーバー起動
make dev-api

# 別ターミナルでフロントエンド起動
make dev-web
```

### デモ用DB（Docker）

```bash
docker compose up -d
```

`dev/seed.sql`は動作確認用の架空のECサイトスキーマ（ユーザー・商品等）で、実データは含まれません。

### ビルド（単一バイナリ）

```bash
make build
./tabgraph
```

## データの保存について

- 接続先DBの認証情報・メタデータは `~/.tabgraph/data.db`（SQLite）にローカル保存されます
- **認証情報は平文で保存されます**。ローカル専用・個人利用を前提とした設計です。共有端末では使用しないでください
